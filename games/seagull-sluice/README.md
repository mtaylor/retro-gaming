# Seagul Sluice

2D top-down Whitley Bay chip defence — built with canvas and sprites from `sprites/`.

## Story

The **seagull boss** has sent minions to steal every chip bag on the prom. You start with your lunch (5 chips). Minions snatch bags from the board and can mug you on contact. Reach the **CHIPPY** goal before you run out of chips.

## Controls

- **WASD** or **arrow keys** — move
- **Space** — restart after game over

## Run locally

```bash
cd /path/to/retro-gaming
python3 -m http.server 8080
```

Open http://localhost:8080/games/seagull-sluice/page.html

## Assets

Sprites are copied from `sprites/henry.zip`, `eleanor.zip`, `seagul-standing.zip` (minions), and `seagul-boss.zip` (HUD boss).
